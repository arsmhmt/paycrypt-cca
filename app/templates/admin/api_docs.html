API Documentation

Welcome to the PayCrypt API documentation. Here you will find information about available endpoints, authentication, and usage examples.

Authentication

All API requests require JWT authentication. Obtain your token via the login endpoint and include it in the Authorization header as Bearer <token>.

Endpoints

  • POST /api/v1/login - Obtain JWT token
  • POST /api/v1/refresh - Refresh JWT token
  • GET /api/v1/payments - List payments
  • POST /api/v1/payments - Create payment
  • GET /api/v1/withdrawals - List withdrawals
  • POST /api/v1/withdrawals - Request withdrawal

Example Request

curl -X GET "https://yourdomain.com/api/v1/payments" \
     -H "Authorization: Bearer <your_jwt_token>"

For more details, contact support or see the full developer documentation.
